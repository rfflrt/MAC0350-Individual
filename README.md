# Infinite Maze

This repository contains the code for the individual project for the first cycle of the MAC0350 discipline of 2026-1 (WebMAC by CodeLab).

## Project Requirements
As for now, the project needs to have
- Front-end comprising HTML, CSS and JS
- Back-end with FastAPI
- Databases

## Maze Generator
This project consists of an infinite maze, with additional mechanics such as enemies and loot.<br>
The maze will be generated by celullar automata instead of graph traversal algorithms, allowing changes to the generation in relation to progress, items and others.

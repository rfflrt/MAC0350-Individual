# MineSweeper 3000

This repository contains the code for the individual project for the first cycle of the MAC0350 discipline of 2026-1 (WebMAC by CodeLab).

## Project Requirements
As for now, the project needs to have
- Front-end comprising HTML, CSS and JS
- Responsivity to mobile screens
- Back-end with FastAPI
- Databases with SQL
- Use of HTMX

## MineSweeper
This project consists of a modified version of the classic MineSweeper game.<br>
The game will have dynamics such as:
- Powers
- Dynamic mines
- Rewards for replaying
- Leaderboards
